# Modern Consumer - Shopify Storefront

A high-converting, minimalist e-commerce storefront built for the modern consumer, designed to feature curated product collections, seamless shopping flows, and an optimized checkout experience.

## 🎯 Why I Built This
I created this project to master e-commerce user experience (UX) engineering and online retail design patterns. My focus was on implementing clean visual layouts that prioritize product discovery, minimizing user friction during the browsing process, and handling complex grid systems for product listing pages.

## ✨ Key Features
* 📦 **Dynamic Product Grid:** A beautifully structured gallery layout featuring clean product cards, clear pricing, and interactive badge overlays (e.g., "Best Seller", "New Arrival").
* 🛒 **Frictionless Shopping Cart Flow:** An intuitive and accessible navigation layout designed to give users immediate feedback when interacting with products.
* 🏷️ **Curated Collections Navigation:** A well-organized filtering layout that allows consumers to effortlessly browse by category, collection, or trend.
* 📱 **Mobile-First Checkout Layout:** Fully optimized for mobile commerce, ensuring that product photos, descriptions, and checkout buttons scale perfectly on smartphones.

## 🛠️ Tech Used
* Shopify Liquid template language & Storefront architecture
* Semantic HTML5 & Schema markup for e-commerce SEO
* Custom CSS3 (Flexbox & CSS Grid for dynamic product layouts)
* Responsive Web Design (Media Queries for cross-device mobile shopping)
